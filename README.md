# Svelte Pocket

## Get Started

- > `npm run install:clean` or `npm install`
- >new class? `npm run build:tailwind` > adds to [stylesheet](src/assets/styles/tailwind.css)
- >`npm run build:fontawesome`
- >`npm run dev` -- [https://localhost:5000](https://localhost:5000)
